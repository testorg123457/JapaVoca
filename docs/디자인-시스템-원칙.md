# 디자인 시스템 원칙 — "AI 디자인 냄새" 회피

> 이 문서는 JapaVoca UI 작업의 **상위 규칙**이다. 개별 화면 디자인은 나중에 또 바뀔 수 있으니,
> **구조/버그를 먼저 단단히** 잡고 그 위에서 디자인 시스템을 절제 있게 다듬는다.
> 색·간격·라운드·타이포는 `apps/mobile/src/theme/tokens.ts` 단일 소스만 사용(하드코딩 금지).

---

## 1. 피해야 할 AI 디자인 냄새 (체크리스트)

1. **색을 거의 하나만 쓰기 — 가장 중요.**
   - 메인이 민트라고 **글자·카드·아이콘·버튼까지 전부 민트**로 칠하지 않는다.
   - 토스/배민도 메인 컬러는 민트·파랑이지만, 글자·카드·보조 요소엔 **중립 그레이/검정**을 적절히 섞는다.
   - **글자색 기본 = 검정**(`text-primary` = gray[900]). 민트 글자는 *활성/선택 상태*나 *주액션 라벨* 같은 **기능적 위치에만**.
   - 민트는 "강조"가 아니라 **기능 신호**다: 주액션·활성 탭·선택됨. 장식으로 쓰지 않는다.

2. **8px grid에 집착하지 않기.** 4/8 배수에 기계적으로 맞추지 말고 눈에 맞는 간격을 쓴다.

3. **너무 완벽한(=사람이 안 만든 것 같은) 마감 지양.** 과하게 균일·대칭이면 오히려 인공적이다.

4. **카드 UI 남용 금지.** 정보는 카드 박스 대신 **텍스트 + spacing + subtle line(헤어라인 구분선)**으로.
   카드는 hero/주액션 진입처럼 **정당한 곳에만**.
   - (적용됨) 홈 "오늘의 학습 현황" → 구분선 인라인 통계, 지갑 거래내역 → 풀폭 구분선 리스트.
   - 설정 → `ListSection`/`ListRow`(풀폭 면 + 헤어라인 + `>`).

5. **micro-interaction는 목적 있는 곳에만.**
   - 버튼 = 즉각 반응(press scale/opacity).
   - 리스트 행 = 거의 없음(눌림 피드백 정도만).
   - **중요한 상태 변화**(보상 공개, 정답/오답)에만 애니메이션.
   - 모든 곳에 비슷한 인터랙션을 깔지 않는다.

## 2. 색 역할(Color Roles) — 무엇을 위한 색인가

| 역할 | 토큰 | 쓰는 곳 | 쓰면 안 되는 곳 |
|---|---|---|---|
| 본문/제목 글자 | `text-primary`(검정), `text-secondary/tertiary`(그레이) | 대부분의 텍스트 | — |
| 브랜드 민트 | `brand` / `bg-brand` | 주액션 버튼, 활성 탭, 선택 상태, 캐시 hero 면 | 일반 글자색, 장식 아이콘, 일반 카드 |
| 옐로/앰버 | `amber` | 캐시·코인·리워드·스트릭 | 그 외 |
| 위험/감소 | `danger` | 로그아웃, 캐시 사용(−), 오답 | 일반 강조 |
| 중립 그레이 | `text-*`, `border-*`, `bg-*` | 아이콘 기본·구분선·면·보조정보 | — |

**원칙:** 한 화면에서 민트가 2~3곳을 넘으면 과하다고 보고 점검한다. 아이콘 기본색은 **중립 그레이**, 민트는 의미 있는 곳만.

## 3. 진행 현황 (2026-06-25)

**팔레트 민트 → Vermilion 전면 교체 완료** (`tokens.ts` 단일 소스):
- Primary = Vermilion `#E8432D`(vermilion-500), 보조 Coral `#FF6B50`(400), 캐시 = 옐로 `#FFCE00`.
- **글자색 = Ink `#1A1A1A`/그레이** 기본. 메인색을 글자색으로 쓰지 않음(브랜드 강조 텍스트만 예외).
- **배경 = 순백 `#FFFFFF`**, 그레이 램프는 핑크 틴트 없는 뉴트럴.
- **탭바 활성색 = Ink(검정)**. 메인색은 hero·주액션 버튼·활성 상태·브랜드 아이콘에만.
- success=green `#2AC171`, danger=vermilion, info=`#3D7FE8`로 기능색 분리(태그 정답/오답 등 색 다양성 확보).
- 카드 절제: 홈 통계 인라인화, 지갑 내역 풀폭 구분선 리스트, 설정 ListSection/ListRow.
